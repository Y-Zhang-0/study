# AI 应用开发 + 后端架构学习路线 V2

> 定制规格：工作日 2h/天 · 周末 8h/天
> **复合型路线**：共 28 周，总计 728 小时
> **目标**：AI 应用开发（主线）+ 后端架构（分布式/高可用/高并发）
> **适用对象**：3.5+ 年全栈开发经验（Java/Node.js/Vue）
> **核心设计**：后端知识融入 AI 学习过程，边做 AI 边学后端

---

## 设计理念

传统学习路线将 AI 和后端分开，导致知识割裂、学了不用、用时忘了。本路线的核心思路是 **在 AI 项目中自然引入后端能力**——当 RAG 系统需要 API 时学 FastAPI，当需要缓存时学 Redis，当需要部署时学 Docker 和 K8s。每个后端知识点都有明确的 AI 项目场景承接，学完即用。
