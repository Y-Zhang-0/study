## 模块 A：大模型推理加速（约 2 周 / 52h，需 GPU）

> 触发补攻信号：目标岗位涉及自建模型推理服务、私有化部署、降本增效；或面试频繁追问推理优化。
> 环境：本地 8GB+ 显存，或云 GPU 按时租用（用完即停，控制成本）。

### A1 推理基础与显存
**学习内容**：
- LLM 推理流程：Prefill 与 Decode 两阶段、自回归生成
- KV Cache 原理：显存占用来源、随序列长度增长
- 显存估算：参数量 × 精度 + KV Cache + 激活值的粗算方法
- 批处理（Batching）对吞吐与延迟的影响

**检查点**：
- [ ] 能估算给定模型 + 上下文长度下的大致显存需求
- [ ] 理解为何长上下文、高并发会撑爆显存
- [ ] 说清 Prefill 与 Decode 的算力/访存特征差异

### A2 vLLM 部署
**学习内容**：
- PagedAttention 原理：显存分页、减少碎片、提升并发
- 连续批处理（Continuous Batching）：动态拼批提升吞吐
- vLLM 部署开源模型，暴露 OpenAI 兼容 API
- 关键参数：`gpu-memory-utilization`、`max-model-len`、张量并行

**检查点**：
- [ ] 用 vLLM 部署一个开源模型并通过 OpenAI 兼容接口调用
- [ ] 对比开启/关闭连续批处理的吞吐差异
- [ ] 把主线项目的 LLM 后端切换到自建 vLLM 服务

### A3 TGI 与量化
**学习内容**：
- TGI（Text Generation Inference）部署与特性，对比 vLLM
- 量化方法：GPTQ / AWQ / INT8，精度与显存/速度的权衡
- 量化模型的部署与质量评估
- 选型：vLLM vs TGI vs Ollama 的适用场景

**检查点**：
- [ ] 部署一个量化模型，对比量化前后显存与质量
- [ ] 能给出 vLLM/TGI/Ollama 的选型结论与理由

### A4 高并发推理与压测
**学习内容**：
- 并发请求处理：请求队列、批处理窗口、超时
- 压测工具（locust/k6/wrk）与推理服务基准
- 关键指标：吞吐（tokens/s）、首字延迟（TTFT）、P95、显存占用
- 调优：批大小、并发度、KV Cache 配置、降级策略

**检查点**：
- [ ] 建立推理服务性能基准（吞吐/TTFT/P95/显存）
- [ ] 通过调参把吞吐或延迟优化到可量化的改善
- [ ] 产出一份"推理优化前后对比"报告（可写进作品集/面试叙事）

**资料**：vLLM 文档、Hugging Face TGI 文档、各量化库（AutoGPTQ/AutoAWQ）文档。

## 模块 B：Go 后端（约 1.5 周 / 39h，可选第二语言）

> 触发补攻信号：目标公司后端技术栈为 Go；或岗位强调高性能网关、Go 生态。
> 已掌握 FastAPI 即满足后端硬性要求；本模块用于在"高性能入口"场景下多一张牌。

### B1 Go 基础
**学习内容**：
- 语法速览：包/函数/结构体/接口/错误处理，对比 Python
- goroutine 与 channel：轻量并发模型
- `sync`、`context`：并发控制与取消传播
- 模块管理（go mod）与项目结构

**检查点**：
- [ ] 能用 goroutine + channel 写一个并发任务示例
- [ ] 理解 Go 错误处理与 `context` 取消机制
- [ ] 说清 Go 并发模型与 Python asyncio 的差异

### B2 Go Web（Gin/Echo）
**学习内容**：
- Gin/Echo 路由、中间件、参数绑定与校验
- JSON 处理、统一错误响应、日志中间件
- 调用 LLM API + 流式响应（SSE/WebSocket）
- 连接池、限流中间件

**检查点**：
- [ ] 用 Gin/Echo 写一个带中间件的 REST 服务
- [ ] 实现 LLM 流式转发（SSE 或 WebSocket）

### B3 Go 高性能 Agent 网关
**学习内容**：
- 用 Go 实现高性能 Agent API 网关（鉴权/限流/路由/流式）
- 网关分工：Go 高性能入口 + Python 业务/Agent 灵活层
- 压测对比 Python 版网关（QPS/延迟/资源占用）

**检查点**：
- [ ] Go 网关与 Python 网关做同口径压测，产出对比数据
- [ ] 能讲清"何时用 Go、何时用 Python"的工程取舍

**资料**：Go by Example、Gin/Echo 官方文档、Go 并发模式资料。
