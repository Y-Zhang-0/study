# 企业级 AI Agent 全栈工程师学习路线 V3

> 定制规格：工作日 2h/天 · 周末 8h/天
> **全栈复合路线**：共 38 周，总计 988 小时
> **目标**：从 0 到 1 独立交付**企业级 AI Agent 应用**——前端交互 + 后端架构 + 数据库 + 大模型集成全流程
> **适用对象**：3.5+ 年全栈开发经验（Java / Node.js / Vue）
> **第一要务**：以最快路径练成 AI 应用开发能力并到达**可投简历门槛**（约 5.5 月），此后边投边学

---

## 核心策略：两阶段 + 求职前置

```
══ Stage A：先具备 AI 应用开发能力 ════════►│门槛│══ Stage B：边投边学 ══════►
P1   P2   P3   P4         P5            │D147│  P6    P7    P8    P9
               ↑软启动(D105)            │可投│
               RAG 全栈上线              │简历│  ★求职正式启动，P6 起边投边学
```

- **Stage A（P1-P5）**：练成"能开发并上线 AI 应用"的核心能力，每阶段产出可部署作品
- **软启动**：P4 末（RAG 全栈应用上线，约 3.5 月）开始零星投递
- **正式门槛**：P5 末（Agent 就绪，约 5.5 月）转全力求职
- **Stage B（P6-P9）**：后端架构、云原生、多模态、综合项目——边面试边补强

## 如何使用本路线

1. **先读** [00-overview.md](./00-overview.md) 建立全局观与两阶段策略，**再用** [daily-schedule.md](./daily-schedule.md) 作每日导航。
2. **Stage A 全力冲门槛**：P1-P5 每阶段产出一个上线作品；P4 末首个 RAG 应用上线即开始零星投递。
3. **P5 末转边投边学**：Stage B（P6-P9）与求职并行，**面试反馈优先于课表**——市场考什么先补什么。
4. 每天按日清单「主题 → 学习内容 → 检查点」推进，当日产出沉淀进作品集并保持线上部署。
5. 每周日做「每周复盘」；每阶段末用 Buffer 做里程碑验收。
6. **加分项（推理加速 / Go）**见附录，达成核心目标后按岗位需求择期补攻，不阻塞主线与求职。
7. 善用 Claude Code / Cursor 加速编码，把精力留给设计、调试与理解。
