# 英语学习 6 个月实施计划

> **执行说明：** 本计划按阶段和周次拆解，每项任务配有验收标准。步骤使用 `- [ ]` 语法跟踪进度。

**目标：** 6 个月内建立日常口语自由交流能力

**方案架构：** 词汇激活（月1-2）→ 语感建立（月3-4）→ 口语产出（月5-6）

**工具清单：** 墨墨背单词（牛津5000词·美版）、ESLPod、Rachel's English（YouTube）、《老友记》前两季、ChatGPT语音

---

## 启动任务：环境配置（第一天，约15分钟）

**目标：** 完成所有工具配置，第一天即可正式开始学习。

- [ ] **步骤1：配置墨墨背单词**
  - 打开墨墨背单词 → 选择词书「牛津英语5000词·美版」
  - 进入「今日计划」→「每日新词数」→ 设置为 **15**
  - 开启「例句模式」（设置 → 学习设置 → 展示例句）
  - 设置每日提醒时间（建议固定早晨或午休）

- [ ] **步骤2：准备ESLPod**
  - 搜索「ESLPod」或访问 eslpod.com，订阅播客或下载App
  - 找到 "English as a Second Language Podcast" 第1集，试听2分钟
  - 验收：能找到音频并正常播放

- [ ] **步骤3：准备Rachel's English**
  - YouTube 搜索「Rachel's English」，订阅频道
  - 找到视频「American English Pronunciation」系列第1集
  - 验收：能正常访问并播放

- [ ] **步骤4：准备《老友记》**
  - 确认能获取《老友记》第一季第1集（流媒体/本地）
  - Phase 2 月4才用，现在只需确认来源
  - 验收：能找到可用视频源（无需现在观看）

- [ ] **步骤5：启动当天墨墨学习**
  - 完成今日第一批15个新词 + 系统推送的复习
  - 验收：墨墨当日任务完成，看到「今日完成」状态

---

## Phase 1：激活期（月1-2，共8周）

**目标：** 激活遗忘词汇，建立高频词快速提取能力，积累900个新词。

**每日安排：** 25-30分钟，全部用于墨墨背单词

**墨墨设置：** 每日新词15个，复习30-50个（峰值日跳过例句）

---

### 第1-2周：建立习惯

- [ ] **每日执行**
  - 完成墨墨当日新词（15个）+ 自动推送复习
  - 复习量超过50个时：跳过例句，仅看翻译，保证在30分钟内完成

- [ ] **第7天检查点**
  - 查看墨墨「学习报告」，记录本周完成天数和平均准确率
  - 验收标准：完成天数 ≥ 5天
  - 若准确率低于70%：说明新词量偏多，将每日新词暂时降至10个

- [ ] **第14天检查点**
  - 查看墨墨「学习报告」，记录累计新词数和本周准确率
  - 验收标准：累计新词 ≥ 180个，本周准确率 ≥ 75%

---

### 第3-4周：稳定输出

- [ ] **每日执行**（同第1-2周）

- [ ] **第21天检查点**
  - 验收标准：累计新词 ≥ 270个，准确率 ≥ 78%

- [ ] **第28天（第1个月末）里程碑**
  - 查看墨墨本月总结
  - 验收标准：累计新词 ≥ 360个，近7天准确率 ≥ 80%
  - 若未达标：本月剩余时间继续强化，不急于进入下月计划

---

### 第5-6周：加速激活

- [ ] **每日执行**（同前）

- [ ] **第35天检查点**
  - 验收标准：累计新词 ≥ 525个，准确率 ≥ 82%

- [ ] **第42天检查点**
  - 验收标准：累计新词 ≥ 630个，准确率 ≥ 83%

---

### 第7-8周：达标冲刺

- [ ] **每日执行**（同前）

- [ ] **第49天检查点**
  - 验收标准：累计新词 ≥ 735个，准确率 ≥ 84%

- [ ] **第56天（Phase 1 结束）里程碑**
  - 验收标准：
    - 累计新词 ≥ 840个
    - **连续7天准确率 ≥ 85%**
    - 新词学习无明显卡顿（每个新词学习时间 < 2分钟）
  - ✅ 全部达标 → 进入 Phase 2
  - ❌ 未达标 → 延长2-4周，继续当前节奏，每2周重新评估一次

---

## Phase 2：语感期（月3-4，共8周）

**目标：** 通过影子跟读建立语音肌肉记忆，达到跟读ESLPod后无需重听即可复述完整句子。

**每日安排：** 墨墨15分钟 + 跟读15分钟 = 合计30分钟

**墨墨设置调整（进入Phase 2 第一天操作）：**

- [ ] **步骤：调整墨墨每日新词数**
  - 进入「今日计划」→「每日新词数」→ 改为 **5**
  - 验收：设置保存成功

---

### 第9-10周（月3前2周）：ESLPod 跟读入门

**跟读方法：** 听一句 → 暂停 → 原音原调复述 → 继续下一句

- [ ] **每日��行**
  - 墨墨复习15分钟（新词5个 + 自动推送复习）
  - ESLPod 跟读15分钟：选1集，每次跟读2-3个对话片段
  - 峰值日（复习量>80个）：墨墨跳过例句，保证15分钟内完成

- [ ] **第63天检查点（第9周末）**
  - 自测：随机挑选一个已跟读的ESLPod片段，听一遍后尝试复述
  - 验收标准：能复述出关键词和句子结构，不要求逐字

- [ ] **第70天检查点（第10周末）**
  - 验收标准：跟读时跟上语速不掉拍，复述率 ≥ 60%

---

### 第11-12周（月3后2周）：ESLPod + Rachel's English

- [ ] **每日执行**
  - 墨墨复习15分钟
  - ESLPod 跟读10分钟
  - Rachel's English 发音训练5分钟（每周选1-2个发音专项视频，本周重复练习）

- [ ] **第77天检查点**
  - 验收标准：跟读ESLPod后无需重听即可复述完整句子（≥ 1句）

- [ ] **第84天（第3个月末）里程碑**
  - 验收标准：
    - ESLPod 跟读后能无重听复述完整句子
    - 听懂率 ≥ 70%
  - 若未达标：延长2周继续ESLPod，不切换老友记

---

### 第13-14周（月4前2周）：引入《老友记》

> 注意：切换老友记后听懂率下降属正常，不代表退步，坚持跟读。

- [ ] **每日执行**
  - 墨墨复习15分钟
  - 老友记跟读15分钟：每次选1个片段，**不超过1分钟**，反复跟读至自然

- [ ] **第91天检查点**
  - 自测：选一个已跟读老友记片段，盲听后尝试复述大意
  - 验收标准：能说出片段主要内容，哪怕用简单句

- [ ] **第98天检查点**
  - 验收标准：老友记1分钟片段跟读后语调不僵硬，不需要刻意停顿

---

### 第15-16周（月4后2周）：老友记强化

- [ ] **每日执行**（同第13-14周）

- [ ] **第112天（Phase 2 结束）里程碑**
  - 验收标准：
    - **跟读ESLPod后无需重听即可复述完整句子**
    - **ESLPod 听懂率 ≥ 80%**
  - ✅ 全部达标 → 进入 Phase 3
  - ❌ 未达标 → 延长2-4周，继续跟读，每2周重新评估

---

## Phase 3：产出期（月5-6，共8周）

**目标：** 能就日常话题连续说3-5句，不卡顿。

**每日安排：** 墨墨15分钟 + 跟读5分钟 + AI对话10分钟 = 合计30分钟

**墨墨设置调整（进入Phase 3 第一天操作）：**

- [ ] **步骤：确认每日新词数维持5个**（Phase 2已调整，保持不变）

**ChatGPT 初始化（第一天）：**

- [ ] **步骤：配置ChatGPT对话伙伴**
  - 打开ChatGPT，新建对话，输入以下Prompt：
    > "You are my English conversation partner. Chat with me in simple everyday English for 5 minutes. Correct me when I make mistakes and explain briefly why."
  - 验收：ChatGPT用英语回复并开始对话

---

### 第17-18周（月5前2周）：AI对话热身

**话题库（由易到难）：** 今天天气 → 周末计划 → 喜欢的食物 → 最近看的电影

- [ ] **每日执行**
  - 墨墨复习15分钟
  - 老友记跟读维护5分钟（选已熟悉的片段）
  - ChatGPT语音对话10分钟（从话题库选1个话题）

- [ ] **第119天检查点 + 第一次录音**
  - 自测：就「今天天气」话题连续说3句，不查词
  - 验收标准：说出3句，即使有语法错误也算通过
  - 录音：录制60秒自说，将文字版发给ChatGPT获取反馈，纳入下周练习重点
  - 转写方式：使用手机语音备忘录转写功能，或在ChatGPT语音模式对话后复制文字

- [ ] **第126天检查点 + 第二次录音**
  - 验收标准：每次对话能主动发起至少3轮来回，不需要ChatGPT一直引导
  - 录音：录制60秒，与第119天对比，重复ChatGPT反馈流程

---

### 第19-20周（月5后2周）：扩展话题

- [ ] **每日执行**（同前，话题扩展到更多日常场景）

- [ ] **第133天里程碑 + 第三次录音**
  - 录音：录制60秒，话题自选（例如：介绍自己的一天）
  - 重复ChatGPT反馈流程，记录问题纳入下周练习重点
  - 验收标准：录音不中断，能说满60秒

---

### 第21-22周（月6前2周）：固化输出

- [ ] **每日执行**（继续扩展话题，针对上周录音问题强化练习）

- [ ] **第140天 + 第四次录音**
  - 重复ChatGPT反馈流程

- [ ] **第147天 + 第五次录音**
  - 与第119天录音对比：语速、流利度是否有提升
  - 重复ChatGPT反馈流程

- [ ] **第154天检查点**
  - 自测：就任意一个日常话题连续说5句，不查词、不停顿超过3秒
  - 验收标准：完成5句连续输出

---

### 第23-24周（月6后2周）：收尾验收

- [ ] **每日执行**（同前）

- [ ] **第161天 + 第六次录音**
  - 话题：介绍自己的日常生活（1分钟）
  - 重复ChatGPT反馈流程

- [ ] **第168天 + 第七次录音（最终验收）**
  - **最终验收测试：**
  - **最终验收测试：**
    - 选3个日常话题（天气、食物、周末计划），每个话题连续说3-5句，不卡顿超过3秒
    - 与ChatGPT完成5轮来回对话
  - 验收标准：
    - ✅ 每个话题能说出3句以上
    - ✅ 对话中不需要对方重复问题
  - 完成后进入**自主维护模式**：每日墨墨复习 + 每周2-3次AI对话，持续巩固
